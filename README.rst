SetUp
--------

Prerequisites:

* Install libsox and libcurl for Wit.ai
* Install PyAudio and Flac for SpeechRecognition

Steps:
 1. In config folder, update API keys in template_api.ini
 2. Change name from template_api.ini --> priv_api.ini
